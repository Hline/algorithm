[문제]
A도시의 어떤 번화가에는 N개의 마트가 N – 1개의 도로로 연결되어 있으며 각 도로는 유동인구와 도로길이 정보를 가지고 있다. 
또한 임의의 마트에서 임의의 마트로 항상 이동할 수 있다. 
이 마트에는 Main road라 불리는 경로가 있는데 임의의 두 마트 연이어 연결한 경로들 중 유동인구가 가장 많은 경로를 뜻한다. 
이 도시의 상권조사를 위해 각 마트에서 Main road까지의 거리를 조사하려고 한다. 
모든 마트의 Main road까지의 거리를 구하여 그 합을 출력하시오. 
위 예시에서는 2 → 1(3), 6 → 5(1), 7 → 5(1+6), 9 → 8(5) 해서 3+1+7+5=16 이다.

[조건]
최대 40개의 Test case에 대하여 C++ : 1.5초, JAVA : 2초
메모리 제한 256MB
3 <= N <= 100,000
1 <= 유동인구, 도로길이 <= 100,000
Main road는 오직 하나 존재한다.
Main road에 포함된 마트은 Main road까지의 거리가 0으로 간주된다.
Main road에 포함되지 않은 마트는 Main road에 포함된 마트 중 가장 가까운 마트까지의 거리가 Main road까지의 거리이다.
입력에는 첫째 줄에 Test Case의 수
그 다음 줄에 마트 수 N
그 다음 줄부터 N - 1줄에 걸쳐 정점 A, B와 유동인구, 도로길이가 차례로 주어진다.

[출력]
#1 16
#2 0
